# Lift Station Sizing Tool v4.0
**Clark Dietz, Inc. - Professional Engineering Design Tool**

## Features
- Multi-point elevation profiles (up to 10 points)
- Smith & Loveless methodology for TDH calculation
- Siphon flow analysis with vacuum calculations
- Professional air valve sizing
- CORRECTED HGL calculation (P=0 at discharge)
- Minor loss components with multiple locations
- Comprehensive export capabilities

## Methodology
Uses industry-standard Smith & Loveless approach:
- TDH calculated to controlling high point only
- Gravity/siphon flow after high point
- HGL calculated backwards from discharge (P=0)
